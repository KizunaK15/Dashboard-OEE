DASHBOARD OEE
=============

Sistem ini memantau efisiensi mesin pabrik Anda secara langsung. Anda mendapatkan data akurat secara seketika tanpa perlu melakukan perhitungan manual.

FITUR UTAMA
-----------
- Pemantauan Langsung: Anda melihat perubahan data ketersediaan dan performa mesin setiap detik.
- Grafik Interaktif: Layar menampilkan tren produksi harian dalam bentuk visual yang jelas.
- Integrasi Firebase: Sistem memperbarui informasi secara otomatis menggunakan database waktu nyata.

METRIK PRODUKSI
---------------
1. Availability: Sistem menghitung persentase waktu aktif kerja mesin.
2. Performance: Sistem menghitung kecepatan proses produksi aktual milik Anda.
3. Quality: Sistem menghitung jumlah produk yang memenuhi standar kualitas utama.

TEKNOLOGI
---------
- HTML5
- CSS3
- JavaScript
- Firebase Realtime Database

LANGKAH PENGGUNAAN
------------------
1. Anda menyalin repositori ini ke dalam komputer lokal.
2. Anda membuat file bernama firebaseConfig.js di folder proyek.
3. Anda memasukkan kredensial database Firebase ke dalam file tersebut.
4. Anda membuka file index.html menggunakan browser internet Anda.
